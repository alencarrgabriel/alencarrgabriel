## Sobre Mim

Sou um profissional da área de TI com sólida experiência em suporte técnico, incluindo manutenção de equipamentos, suporte a redes e administração de sistemas operacionais como Windows e Linux. Tenho um perfil proativo, focado em solucionar problemas e entregar soluções eficientes.

Atualmente, sou estudante de **Análise e Desenvolvimento de Sistemas** no **Uniceub**, onde estou aprofundando meus conhecimentos em desenvolvimento de software e tecnologias relacionadas. Estou em transição para a área de desenvolvimento, buscando aplicar meu conhecimento técnico em projetos desafiadores e inovadores. Meu objetivo é me tornar um programador profissional, especialmente utilizando tecnologias como **.NET**, **Blazor**, **Flutter** e **NestJS**.

Estou em busca de uma oportunidade para contribuir com meu conhecimento, aprender novas tecnologias e evoluir profissionalmente.

## Projetos

- **[Gerador de Recibos](https://github.com/alencarrgabriel/GeradorRecibos)**: Aplicação desktop offline para geração de recibos em PDF, controle de gavetas de caixa e gestão financeira operacional. Desenvolvida com **Python**, **PySide6** (Qt 6), **ReportLab** e **SQLite**, seguindo princípios de **Clean Architecture**. Inclui sistema de autenticação com PBKDF2, backup automático, temas claro/escuro e suporte a operação em rede.

- **[Feliz no Simples](https://github.com/alencarrgabriel)**: Plataforma completa com app mobile (**Flutter 3** + **Dart**) e backend (**NestJS** + **TypeScript**) seguindo Clean Architecture. Utiliza **PostgreSQL** com **PostGIS** para dados geoespaciais, **Redis** para cache e sessions, **Socket.IO** para chat em tempo real, **Firebase Auth** para login social e **Docker** para containerização. Pipeline CI/CD automatizado com **GitHub Actions**.

- **[Portfólio Pessoal](https://alencarrgabriel.github.io/)**: Site desenvolvido com **HTML**, **CSS**, **JavaScript** e **Bootstrap** para apresentar projetos, habilidades e trajetória profissional.

- **Aplicação de Inspeção Veicular**: Sistema desenvolvido com **Blazor** e **.NET MAUI** para realizar inspeções em caminhões, registrando informações como calibragem de pneus, hodômetro e condição geral do veículo.

- **Site de Vendas**: Aplicação web com operações CRUD, incluindo cadastro de produtos, clientes, pedidos e controle de estoque. Utiliza **ASP.NET Core**, **Blazor Web Assembly** e **SQL Server**.

- **[Dashboard de Vendas](https://github.com/alencarrgabriel/DashboardVendas)**: Projeto utilizando **Python** com **Pandas**, **Plotly Express** e **Streamlit** para análise de dados de vendas. Dashboard interativo com visualização de faturamento por dia, tipo de produto, cidade, forma de pagamento e avaliação das filiais.
